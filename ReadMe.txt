Перед установкой программы выполните следующую команду:
pip install -r requirements.txt

По умолчанию программа считывает текст из всех тегов <p>.
Если вы хотите изменить шаблон считывания:
1) Откройте файл "template_example.txt"
2) Создайте файл "template.txt" и поместите в него шаблон согласно формату

Для запуска скрипта используйте:
python main.py <link>
